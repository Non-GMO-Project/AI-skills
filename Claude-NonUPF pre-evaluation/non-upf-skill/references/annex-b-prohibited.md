## Common Alias Lookup

When encountering these on ingredient lists, flag them:

| Label Term | Prohibited As |
|---|---|
| Xanthan gum | Prohibited — industrial fermentation (Section 6.3.4.2) |
| Gellan gum | Prohibited — industrial fermentation |
| Maltodextrin | Prohibited — enzymatic hydrolysis |
| Carboxymethylcellulose (CMC) | Prohibited — chemically modified cellulose |
| Methylcellulose | Prohibited — chemically modified cellulose |
| HPMC | Prohibited — chemically modified cellulose |
| Monk fruit extract | Prohibited — non-nutritive sweetener (Section 6.3.3.3) |
| Erythritol | Prohibited — non-nutritive sweetener |
| Sorbitol | Prohibited — non-nutritive sweetener |
| Xylitol | Prohibited — non-nutritive sweetener |
| Stevia extract / Reb A | Prohibited — steviol glycosides |
| Caramel color (I-IV) | Prohibited — explicitly listed |
| Natural colors (if derived from prohibited sources) | Flag for clarification |
| Modified food starch | Clarify — aluminum starch is prohibited; others may not be |
| Refined oil (unspecified) | Clarify processing method — solvent extracted is prohibited |
